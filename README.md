# Focus Shield Simple

集中モード中に気が散るサイトをブロックする Chrome 拡張機能です。

## インストール

1. このリポジトリをクローンまたはダウンロード
2. Chrome で `chrome://extensions` を開く
3. 右上の「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このプロジェクトのフォルダを選択

## 使い方

### ON / OFF
ツールバーの Focus Shield アイコンをクリックすると集中モードの ON/OFF が切り替わります。ON の間はバッジに「ON」と表示されます。

### ブロックドメインの管理
拡張機能の**オプションページ**からブロック対象のドメインを管理できます。

オプションページの開き方:
- `chrome://extensions` → Focus Shield の「詳細」→「拡張機能のオプション」
- または拡張アイコンを右クリック →「オプション」

オプションページでは:
- ブロック中のドメイン一覧の確認
- 新しいドメインの追加
- 既存ドメインの削除
- デフォルト設定へのリセット
- ブロック回数の確認・リセット

### デフォルトのブロック対象
- `youtube.com`
- `twitter.com`
- `x.com`

デフォルト設定は `config.json` で定義されています。

## ファイル構成

```
├── manifest.json       # 拡張機能マニフェスト
├── config.json         # デフォルトのブロックドメイン設定
├── service_worker.js   # バックグラウンド処理（ブロック判定・状態管理）
├── blocked.html        # ブロック画面
├── blocked.js          # ブロック画面のスクリプト
├── options.html        # オプションページ
├── options.css         # オプションページのスタイル
├── options.js          # オプションページのスクリプト
└── README.md
```
